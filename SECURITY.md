# Security

## Supported Versions

AccelScope is experimental. Security fixes target the latest `main` branch until the project starts
publishing tagged releases.

## Reporting

Please do not open public issues for security-sensitive reports. Contact the maintainer privately
once a public repository owner is assigned.

## Scope

AccelScope downloads benchmark models and executes local inference through installed runtimes. Be
careful with untrusted models and paths, especially when adding new downloader sources.
